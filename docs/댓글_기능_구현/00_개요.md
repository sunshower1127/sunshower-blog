# giscus 댓글 기능 추가 - 개요

## 목표
Astro 블로그에 GitHub Discussions 기반 댓글 시스템인 giscus를 추가합니다.

## giscus란?
- **GitHub Discussions를 댓글 시스템으로 활용**하는 오픈소스 도구
- GitHub 계정으로 OAuth 인증 (별도 회원가입 불필요)
- 모든 댓글이 GitHub repository의 Discussions 탭에 저장됨
- 마크다운, 코드 하이라이팅, 리액션(👍❤️), 답글 기능 지원
- 다크모드 지원, 테마 자동 동기화 가능
- **완전 무료, 광고 없음**

## 현재 상태
- ✅ Disqus 컴포넌트가 설치되어 있지만 **비활성화** 상태
  - 파일: [src/components/Disqus.astro](../../src/components/Disqus.astro)
  - 설정: [src/data/disqus.config.ts](../../src/data/disqus.config.ts) - `enabled: false`
- ❌ giscus는 아직 설치되지 않음

## 왜 giscus인가?
| 항목 | Disqus | giscus |
|------|--------|--------|
| 비용 | 무료(광고 포함) / 유료($9/월~) | 완전 무료 |
| 로그인 | Disqus 계정, 소셜 로그인 | GitHub 계정 |
| 데이터 저장 | Disqus 서버 | 내 GitHub repo |
| 광고 | 무료 플랜에 포함 | 없음 |
| 타겟 | 일반 블로그 | 기술/개발자 블로그 |
| 데이터 소유권 | Disqus 소유 | 사용자 소유 |

## 구현 범위
1. **GitHub 설정** (사전 작업)
   - GitHub Discussions 활성화
   - giscus 앱 설치
   - 설정값 획득

2. **코드 구현**
   - giscus 설정 파일 생성
   - giscus 컴포넌트 생성
   - 블로그 포스트 페이지에 통합
   - 다크모드 테마 동기화

3. **테스트 및 검증**
   - 로컬 빌드/실행 테스트
   - GitHub Discussions 연동 확인
   - 프로덕션 배포 및 테스트

## 예상 소요 시간
- GitHub 설정: 10-15분
- 코드 구현: 15-20분
- 테스트: 10분
- **총 35-45분**
